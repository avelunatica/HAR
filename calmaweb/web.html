Web de administración

Introducción

Creouse unha web para o uso dos administradores, para poder administrar a agregación, borrado ou modificación dos usuarios. Primeiramente a web ten unha explicación de que é CalmaTEA e o transtorno de espectro au tista e un login con usuario e contrasinal no que únicamente se poden rexistrar os administradores, aínda que futuramente poderíase engadir o resto de tipos de usuario.

Configuracións

Unha vez rexistrado, un administrador pode facer 3 accións principais:

  • Xestión de titores: Onde se pode crear un novo tipo de titor, modificarlle algún dos seus parámetros (incluído a contrasinal se é necesario) ou borralo definitivamente do sistema.
  • Xestión de TEA: Similar ó anterior punto, pero centrado nos usuarios con TEA e os seus parámetros específicos.
  • Relación Titor‑TEA: Nesta configuración enlázanse os diferentes titores cos seus usuarios TEA correspondentes.

Toda a información da base de datos se conversa cunha API REST. É por iso que no prototipo esta web está en local: xa que a base de datos tamén está en local.
